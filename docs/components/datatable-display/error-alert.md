# ErrorAlert

Shows an **error alert** when the server query fails (used in server table flows).

[`src/components/DataTable/display/ErrorAlert.tsx`](../../../src/components/DataTable/display/ErrorAlert.tsx)

## Import

```tsx
import { ErrorAlert, type ErrorAlertProps } from '@bsol-oss/react-datatable5';
```

## API

### `ErrorAlertProps`

| Name          | Description                                         |
| ------------- | --------------------------------------------------- |
| `showMessage` | Whether to show the error message (default `true`). |
